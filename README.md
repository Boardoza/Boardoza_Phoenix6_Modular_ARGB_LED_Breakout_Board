# Boardoza Phoenix6 – Breakout Board

The **Boardoza Phoenix6** is a compact, circular **addressable RGB lighting breakout board** featuring **six RGB LEDs** arranged in a ring configuration. Each LED is individually addressable, allowing precise control over color and brightness for dynamic lighting effects.

With its modular and chainable design, multiple B-Phoenix6-BRK boards can be connected to create custom lighting patterns, symbols, or visual indicators. Supporting **both 3.3V and 5V logic levels,** the board is fully compatible with common microcontrollers and development platforms. This makes it a reliable choice for DIY electronics, decorative lighting, wearables, and interactive display applications, whether for prototyping or finished projects.

## Key Features

- **6x Addressable RGB LEDs:** Each LED can be individually controlled for color and brightness, enabling dynamic lighting effects and smooth animations.  
- **Modular Daisy-Chain Design:** Multiple boards can be connected to build custom lighting patterns and effects.
- **Dual Voltage Compatibility:** Operates with both 3.3 V and 5 V power supplies.
- **Logic Level Support:** Fully compatible with **TTL and CMOS** logic systems.
- **Flexible Connection Options:** Includes both **SMD and Through-Hole (THR) pads** for input and output connections.
- **Creative Application Ready:** Suitable for interactive lighting, wearables, decorative projects, and visual indicators.

## Technical Specifications

**Model:** Phoenix6  
**Manufacturer:** Boardoza  
**Function:** Individually Addressable RGB Lighting  
**Input Voltage:** 3.3 V / 5 V  
**LED Model:** T3A33BRG-H9C0002X1U1930   
**Number of LEDs:** 6  
**LED Type:** Individually addressable RGB LEDs  
**Logic Compatibility:** TTL / CMOS  
**Data Interface:** Serial Data & Clock (DIN / CIN / DOUT / COUT)  
**Daisy-Chain Support:** Yes  
**Operating Temperature:** -40 °C to +85 °C  
**Board Dimensions:** 31.79 mm × 31.71 mm
